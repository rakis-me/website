☃️ About me

Hello, my name is Rakis, I am a passionate web developer, that enjoys teaching how to program. Learning to code is not easy, but it does not have to be a chore either.

I learned to code at a very young age, but that does not mean that you have to follow the same path I did, you can learn anything at any time in life. It does take time, but you don't have to spend all of the time in one season. If you are interested in learning to code spend some time on my site, I am sure you will learn something.

💻 Why learn to code?

With code, you can:

  • 🌎 Communicate with the world
  • 🧱 Build applications
  • 🤑 Provide value
  • 🎨 Create

⛑ How to learn to code?

There are many ways to learn, but it does not have to complex. Here is a path I recommend:

Learn

  • a little HTML/CSS
  • the command line
  • about networking and servers part 1 serve your webpage
  • programming basics with Javascript
  • about networking part 2, APIs
  • to use data and storage
  • to build applications
  • authentication
  • security
  • to learn

These topics should get you started but remember this journey is never ending, so you will always need to learn to learn. You can find many resources about these topics online and you should connect to as many as possible. My goal is to keep things simple, no certificates, no logins, no fancy graphics or animations. If you need to gamify your learning or need faux motivation you can find it else where.

I may start to do some videos of the workshops, but most of material will be pure text. Keep it simple! You will find when you remove all the blinking lights, you see the fundamentals and once you understand the fundamentals then you can create anything!
